# bus-net

Messaging building blocks for .NET — CQRS command and event buses and message
abstractions over pluggable transports (RabbitMQ, Kafka) — each scoped to a single
concern and shipped independently on NuGet under `JorgeCostaMacia.Bus.*`.

> Part of the `JorgeCostaMacia.*` family, on top of the [shared-net](https://github.com/JorgeCostaMacia/shared-net)
> DDD foundation, alongside [http-net](https://github.com/JorgeCostaMacia/http-net) (ASP.NET Core).
